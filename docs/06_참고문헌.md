# 6. 참고문헌

본 보고서 2~5장에서 참조한 교리·표준·프레임워크·기술 스택·학술 자료를 한곳에 모았다. 본문은 가독성을 위해 각주 없이 서술하고, 근거는 이 페이지에 일괄 정리한다. 저장소 링크(`uav-sim-env`, `fried-pollack-ai`, `pollack-ai`)는 입증 자료로서 본문에도 남겨 두었다.
